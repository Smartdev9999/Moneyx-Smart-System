

## แผนปรับปรุง: Light Theme สีนุ่มนวลตา

### ปัญหา
Light Theme ปัจจุบันใช้สีขาวล้วน (100% white) ทำให้สว่างเกินไปและเจ็บตา

### การแก้ไข

**ไฟล์:** `src/index.css`

ปรับ CSS Variables ใน `:root` (Light mode) ให้ใช้สีเทาอ่อนนวลตาแทนสีขาวล้วน:

| Variable | ค่าเดิม | ค่าใหม่ | คำอธิบาย |
|----------|---------|---------|----------|
| `--background` | `0 0% 100%` (ขาวล้วน) | `220 20% 97%` (เทาอมฟ้าอ่อน) | พื้นหลังหลัก |
| `--card` | `0 0% 100%` | `0 0% 100%` | คงสีขาวเพื่อแยกความต่างจาก background |
| `--popover` | `0 0% 100%` | `0 0% 100%` | คงสีขาวสำหรับ dropdown |
| `--muted` | `210 40% 96%` | `220 20% 94%` | พื้นหลัง muted อ่อนลง |
| `--chart-bg` | `210 40% 98%` | `220 20% 95%` | พื้นหลัง chart นุ่มนวล |
| `--sidebar-background` | `0 0% 100%` | `220 20% 98%` | Sidebar นุ่มนวล |

### Color Palette ใหม่ (Light Mode)

```text
Background (main)     : #F5F6FA - เทาอมฟ้าอ่อน (นุ่มนวลตา)
Card/Popover          : #FFFFFF - ขาว (ตัดกับ background)
Muted                 : #EBEDF3 - เทาอ่อน
Chart Background      : #F0F2F7 - เทาอมฟ้า
Sidebar               : #F8F9FB - เทาอ่อนมาก
```

### ผลลัพธ์
- พื้นหลังนุ่มนวลตากว่าเดิม
- Cards ยังคงโดดเด่นบนพื้นหลัง
- ความคมชัดระหว่าง elements ยังคงดี
- ลดความเมื่อยล้าของสายตา
